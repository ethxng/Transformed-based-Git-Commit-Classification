# Transformed-based-Git-Commit-Classification
We use three different methods at evaluating the feasibility of classifying Git Commit messages: TF-IDF + Logistic Regression, fine-tuning standard CodeBERT, and fine-tuning CodeBERT + Inverse Class Frequency (to address serious class imbalance in the dataset). Our dataset comes from CommitBench on HuggingFace.
